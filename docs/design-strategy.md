# Design Strategy

## Web

Use the vault reference: `workspace-assets/designs/apple-style-reference.md`.

Apply it as a direction, not as a brand clone.

- Canvas: #f5f5f7
- Surface: #ffffff
- Text: #1d1d1f
- Secondary text: #707070
- CTA: #0071e3, used sparingly
- Large question-led hero
- Quiz interaction visible above the fold
- No heavy box shadows
- Spacious layout on desktop
- Mobile-first quiz usability

## Apps in Toss

Later Apps in Toss implementation should follow TDS and App Builder guidance.

Do not reuse Toss/TDS assets on the standalone web unless allowed.
