# Product Brief

## Name

- Korean: 센스있게, 예의있게
- English: Sense & Manners
- Apps in Toss appName: sense-and-manners

## One-liner

한국 사회생활에서 헷갈리는 예의와 센스를 퀴즈로 익히고, 필요한 순간에는 빠르게 확인하는 서비스.

## Target Users

- 한국인 사회 초년생
- 한국 문화가 익숙하지 않은 외국인 또는 유학생

Both groups may not know implicit etiquette in Korean social situations.

## First Scope

- 회사 생활
- 식사 예절
- 장례식장

## Primary UX

Quiz-first.

The first screen should lead with `나는 예의 있는 사람일까?` and a clear start action.

## Quiz Rules

- First version uses 5 questions.
- Score is calculated internally.
- Users see a result title and friendly feedback, not a numeric score.
- If the selected answer is wrong, a lightbulb button becomes active.
- Pressing the lightbulb reveals the explanation.
- Correct answers can show explanation directly or still allow lightbulb reveal, depending on UX clarity.

## Result Bands

- 90-100: 예의의 달인
- 70-89: 센스 좋은 사람
- 40-69: 조금만 알면 더 자연스러워요
- 0-39: 지금부터 익히면 충분해요
